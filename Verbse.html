Exercise 1

                    मैं = నేను

                    हम = మేము, మనము

                    तू = నీవు

                    तुम = నీవు, మీరు

                    आप = తమరు

                    वह = అతడు, ఆమె, అది, ఆ

                    यह = ఇతడు, ఈమె, ఇది, ఈ

                    वे = వారు, అవి, ఆ

                    ये = వీరు, ఇవి, ఈ

                    हूँ। = ఉన్నాను.

                    हो। = వున్నావు.

                    है। = వున్నది, ఉన్నాడు.

                    हैं। = వున్నవి, వున్నారు.

    1. मैं लडका हूँ। = నేను బాలుడను.

    2. हम आदमी हैं। = మేము మనుష్యులము.

    3. तुम लडकी हो। = నీవు బాలికవ.

    4. आप कौन हैं? = తమరు ఎవరు?

    5. वह किताब है। = అది పుస్తకము

    6. यह दवात है। = ఇది సిరాబుడ్డి.

    7. वे क्या हैं? = అవి ఏమిటి?

    8. ये नक्षत्र हैं। = ఇవి నక్షత్రములు.

    9. कुत्ता कहाँ है? = కుక్క ఎక్కడ వున్నది?

    10. दीवार पर बिल्ली है। = గోడ మీద పిల్లి వున్నది.
